Think Like a Programmer

Pretend you are writing a plugin that automatically harmonizes with MIDI input. What instructions would you give a computer to do this programmatically?
I would create two data inputs for the program; one being the root MIDI to harmonize to, the other being how many semitones away should the plugin add the harmonizing note. When a number for the former is entered, it would be stored as a variable. When a MIDI note is recieved, it will output that number twice. The first time will be outputed as normal to a MIDI playback engine, the second we have the the stored number of the semitone added to it, then outputted with the original MIDI note.
